Wonder Station

Elemento Información
Nombre del proyecto Wonder Station
Descripción - Juego multijugador (máx. 10 personas) para explorar y divertirte con amigos mientras superas retos y dificultades.
- Estilo anime, personaje personalizable y habilidades que se obtienen conforme avanzas en la historia. Algunas habilidades serán exclusivas mediante pago.
Objetivo principal Diversión y entretenimiento
Tecnologías previstas Motor gráfico Unreal Engine 5, base de datos PostgreSQL, servidor en Node.js para gestión de partidas, integración futura con servicios en la nube para partidas privadas.
Fecha de inicio 21/11/2025
Estado actual Planificación
Equipo Actualmente solo yo
Características adicionales - Sistema de temporadas con recompensas.
- Zonas especiales tipo “distrito social” donde los jugadores pueden juntarse sin misiones.
- Eventos semanales con desafíos cooperativos.
- Sistema de progresión inspirado en juegos como Honkai: Star Rail.
- Mercado interno para intercambiar objetos cosméticos.
